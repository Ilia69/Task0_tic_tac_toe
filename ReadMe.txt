Игра крестики-нолики

Состояние поля передаётся в виде списка game_field.
Этот список хранит символы, которые игроки поставили на поле.
Каждый элемент отвечает за одну из 9 клеток поля.
Последний элемент содержит число пустых клеток поля.

Ходы игрока вводятся с клавиатуры. Ходы компьютера случайные.

Сценарий игры:
1. Запуск, сосздание и вывод пустого поля.
2. Игроку предлагается сделать первый ход.
	2.1 Если он соглашается, то нужно ввести клетку для хода
	2.2 Если он отказывается, то компьютер делает первый ход
3. Запускается цикл, в котором игрок и компьютер ходят по очереди.
После каждого хода идёт проверка, есть ли три одинаковых символа
в ряд на поле.
4. Либо после одного из ходов появляется победитель,
либо кончаются свободные клетки поля. Игра заканчивается.
